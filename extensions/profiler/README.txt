CONTENTS
========

This package contains the NetLogo profiler extension.

We thank Roger Peppe for his contributions to the code.

INSTRUCTIONS
============

The profiler extension is pre-installed in NetLogo. For 
instructions on using it in your model, and for more 
information about NetLogo extensions, see the NetLogo 
User Manual.

For a model that demonstrates this extension, see Code
Examples/Profiler Example in the models library.
